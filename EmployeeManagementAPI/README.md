# Employee Management API

## Features
- CRUD Operations
- Repository Pattern
- SQL Server Integration
- Logging with Serilog

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

## API Endpoints
- GET /api/employees
- GET /api/employees/{id}
- POST /api/employees
- PUT /api/employees/{id}
- DELETE /api/employees/{id}
